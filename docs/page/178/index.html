Febest выпустил новые запчасти для Toyota
Бренд Febest расширил фирменный ассортимент за счёт новых деталей для автомобилей Toyota.
Восемь новых позиций уже доступны для заказа, сообщает пресс-служба бренда.
Для Toyota RAV4 представлен сайлентблок с артикулом TAB-731, задняя тяга стабилизатора с номером 0125-AXAA52LOWR и уплотнительное кольцо соленоида управления фазовращателем.
Внутренний ШРУС с артикулом 0111-ZRE142 подходит для Toyota Matrix, а шаровая опора 0120-XU70 и задняя стойка стабилизатора 0123-XU70R – для Toyota Highlander.
Бренд также выпустил новую ступицу и втулку стабилизатора для Toyota Corolla. Детали можно найти по артикулам 0182-ZRE212F и TSB-ZRE212F соответственно.
Гарантия на перечисленные новинки составляет 2 года или 60 000 км в зависимости от того, что наступит раньше.
