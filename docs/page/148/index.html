Carville Racing представил пасту для очистки рук
Бренд Carville Racing объявил о расширении ассортимента автохимии.
В каталоге стала доступна паста для очистки рук с дозатором объёмом 3 литра, сообщает пресс-служба бренда.
Новое средство предназначено для удаления въевшихся загрязнений с кожи рук, включая сажу, водостойкие смазки и краски.
Входящий в состав пасты абразив повышает эффективность очистки; благодаря увлажняющим компонентам такое средство не сушит кожу рук при ежедневном использовании, отмечают представители бренда.
Новую пасту можно найти по артикулу S7003000; её стоимость составляет 3250 рублей.
