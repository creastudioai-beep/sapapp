В ассортименте Miles появилось 18 датчиков износа тормозных колодок.
Новые позиции уже доступны для заказа, сообщает пресс-служба бренда.
Деталь с артикулом EFBS000 произведена для Mercedes-Benz W169, 245 и 203, а с номерами EFBS014 и EFBS018 – для Mercedes-Benz Sprinter и Vito.
Для BMW X5(E70) и X6(E71) выпущен датчик износа тормозных колодок с артикулом EFBS013, для BMW F20, 21, 30, 31 – с номером EFBS019.
Другие новинки подходят для Audi, Volkswagen и Porsche.
Датчики Miles изготавливаются в соответствии со спецификациями автопроизводителей; компоненты и сырье проходят программу проверки в соответствии с международной системой управления качеством IATF16949, отмечают представители бренда.
