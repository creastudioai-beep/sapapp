⛔️ Когда проезд под кирпич допускается Проезд под ...

⛔️
Когда проезд под кирпич допускается
Проезд под знак 3.1 ("Въезд запрещён") разрешен в нескольких случаях:
🟢
Водителям маршрутного транспорта (автобусы, трамваи, маршрутки, троллейбусы), если их маршрут проходит через зону, где установлен знак.
🟢
Если есть табличка с указанием времени действия, то проезд под знак запрещён только в указанное время. Например, если действие знака ограничено по будням с 8:00 до 17:30, в остальное время (по вечерам или выходным) проезд будет разрешён.
🟢
Если знак закрывает только определённую полосу движения, обозначенную табличкой со стрелкой вниз, то проезд по остальным полосам разрешён.
