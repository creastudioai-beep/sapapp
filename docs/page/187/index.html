Россиян могут начать штрафовать каждый день за езду без страховки — по госномеру через дорожные камеры.
Масштабную инициативу готовят совместно МВД, Центробанк и Российский союз автостраховщиков. Госдума может рассмотреть её уже в начале августа — в завершении весенней сессии.
