Компания Tesla отзывает с американского рынка 1,85 млн электромобилей.
Причиной стал риск сбоя в софте, который отслеживает закрытие и открытие капота. Во время движения капот может незаметно для водителя открыться и полностью перекрыть обзор, что может привести к аварии.

В середине июня для устранения проблемы Tesla выпустила обновление программного обеспечения. Обновлённое ПО обнаруживает открытый капот и предупреждает об этом водителя.
В декабре Tesla отозвала 2,03 млн электромобилей в США из-за проблем в системе автопилотирования.
