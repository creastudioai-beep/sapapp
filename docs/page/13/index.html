Бренд Eisen выпустил новые фильтры для грузовиков КАМАЗ, JAC, Dongfeng, Sany и FAW.

Топливный фильтр тонкой очистки с артикулом EIF0101228 подходит для Sany D13, а топливный фильтр-сепаратор с номером EIF0101229 – для ряда моделей Sany.

Компания самостоятельно производит нетканый материал для фильтров кабины; для остальных используется фильтровальная бумага европейских (Ahlstrom и H&V) и локальных марок (Chentai и Renfeng).
