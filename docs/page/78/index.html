Британский бренд, принадлежащий индийской Tata Motors, ранее приостановил экспорт для оценки новых тарифных условий. Несмотря на ограничения, JLR называет США ключевым рынком.
Президент Трамп пообещал смягчить меры, исключив из пошлин часть автокомпонентов — в перечне более 300 позиций.
