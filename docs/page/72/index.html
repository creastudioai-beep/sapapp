Анонсированы тормоза для горных велосипедов, но, как уточняет The Drive, на первом этапе будет выпущено лишь ограниченное количество комплектов для профессиональной маунтинбайк-команды Specialized Gravity. Решение о более широком партнёрстве с производителями велосипедов или запуске продаж компонентов для самостоятельного тюнинга будет принято позже.

Компактные суппорты фирменного красного цвета имеют по четыре поршня диаметром 18 миллиметров и сочетаются с фиксированными дисками, спроектированными с учётом экстремальных торможений во время гонок.
Особо отмечается, что система имеет широкие возможности настройки под предпочтения конкретного райдера. Сотрудничество с фирмой Specialized в Brembo назвали стратегическим: в компании рассчитывают, что оно приведёт к дальнейшим инновациям. Издание уточняет, что на гоночных велосипедах Specialized, которые использует заводская команда, задействованы и другие компоненты из автомобильного мира — например, подвески Ohlins.
