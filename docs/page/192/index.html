Новые зимние шины Pirelli доступны для нового BMW 7 Series с апреля нынешнего года.
Впоследствии они станут устанавливаться и на другие автомобили баварской марки. Во втором полугодии 2024 года они выйдут на рынок в версии для кроссовера X3.
