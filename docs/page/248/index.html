Mercedes-Benz G-класса готовится к обновлению

Появились подробности о плановом обновлении внедорожника Mercedes-Benz G-класса, текущее поколение выпускается с 2018 года. Изменения во внешности и интерьере будут едва заметными.

Главной технической обновкой станет новый двигатель для версии G 500: на смену битурбомотору V8 4.0 (422 л.с.) придет рядная 3.0 литровая турбошестерка схожей мощности.

Двигатель V8 будет предложен только варианту AMG. Также все моторы (включая дизели) обзаведутся 48-вольтовым стартер-генератором.

Дебют обновленного семейства ожидается через год, одновременно будет представлен электрический Mercedes-Benz EQG.

Новый способ защиты масла от подделки

Завод «Юнайтед Петрокемикалс», выпускающий в Калуге моторное масло под маркой Rolf, внедрил на производстве новый метод защиты масла от подделки. В моторное масло добавляют особый краситель, делающий жидкость зеленой.

Для добавления используется особый маслорастворимый краситель, стойкий к окислению и действию ультрафиолета и не вступающий в реакцию с компонентами масла.
