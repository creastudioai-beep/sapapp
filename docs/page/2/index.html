Бронированный S-Class не будут продавать китайцам, американцам, диктаторам и авторитетам — это официальная позиция Mercedes-Benz.
Компания хочет изменить ассоциацию модели с транспортом для лидеров противоречивых политических режимов и бандитов.
Продавать машины «крупнейшим мировым сверхдержавам» тоже не хотят — поэтому клиентов из КНР и США рассматривать не будут.
Для остальных же введут строгую проверку биографических данных. Только после неё можно будет обсудить покупку автомобиля.
