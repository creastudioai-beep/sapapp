Новинки предназначены для популярных в России автомобилей японских и европейских марок.
В частности, владельцам пикапов Mitsubishi L200 адресованы передний карданный вал в сборе (арт. ASM-PSFKL6) и ремонтное решение, которого нет в оригинале, в виде задней крестовины (арт. ASM-KL6R) – в дополнение к передней (арт. ASM-KL6F), которая появилась в ассортименте ранее.
Для автомобилей BMW подходят подвесные подшипники с артикулами BMCB-E60 (для «пятерки» в кузове E60) и BMCB-G20 (для «трешки» в кузове G20).
Кроссовер Kia Sorento можно отремонтировать с использованием подвесного подшипника с артикулом KCB-MQ4, который позволяет не менять карданный вал целиком, как подразумевает автопроизводитель.
Такое же ремонтное решение (подвесной подшипник TCB-XA50) разработано для ряда моделей Toyota.
