Компенсация по ОСАГО при ДТП может быть больше положенной по закону

Компенсация по ОСАГО при ДТП может быть больше положенной по закону
Страховщикам придётся заплатить владельцам пострадавших в ДТП авто сумму, превышающую установленные законом лимиты, если они не направят машину в ремонт и не оплатят его.
Такое внезапное решение принято местными судами и поддержано Шестым кассационным судом.
Согласно закону об ОСАГО, страховщик обязан организовать и оплатить восстановительный ремонт поврежденного автомобиля. Однако, есть исключения, главное из которых — соглашение между страховщиком и пострадавшим о выплате вместо ремонта.
Экспертный совет службы финансового уполномоченного согласился с позицией судов и рекомендовал рассматривать сумму взыскания не как страховое возмещение, а как убытки по Гражданскому кодексу.
Убытки должны быть рассчитаны как действительная стоимость ремонта по рыночным ценам в субъекте РФ, с учётом утраты товарной стоимости и без учета износа на момент рассмотрения спора, а также без ограничения по лимиту страхового возмещения по ОСАГО.
