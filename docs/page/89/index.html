Nissan запатентовал систему снижения выбросов для гибридов
Nissan зарегистрировал патент на усовершенствованную систему изменения фаз газораспределения (VVT) для гибридов с последовательной схемой e-Power. Новая технология снижает выбросы при повторном запуске ДВС.

Технология может применяться в параллельных и подключаемых гибридах, а также в ДВС с системой «старт-стоп».
