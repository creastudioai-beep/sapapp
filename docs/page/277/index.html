Машины с поддельным техосмотром начали снимать с регистрации в ГИБДД

▪️По словам председателя Союза операторов технического осмотра Максима Бурдюгова, владельцев автомобилей с прекращенной регистрацией по причине фальсификации отправляют на повторное прохождение техосмотра — уже по всем правилам.
