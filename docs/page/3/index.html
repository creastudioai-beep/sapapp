В каталоге появились четыре набора, в каждом из которых по 10 предохранителей.

Все ножевые модели имеют стандартизированное цветовое исполнение корпуса в соответствии со стандартом ISO 8820 для быстрой идентификации номинала, отмечают в Miles.
Также для заказа доступны цилиндрические предохранители, предназначенные для более старых моделей машин (в том числе для ВАЗовской классики). Артикул набора с номиналами 5, 8, 16, 20, 25 А – BLF2900.
