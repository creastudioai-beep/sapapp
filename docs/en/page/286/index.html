Комплект для частичной замены масла Ford Mustang в АКПП 10R80. Комплект применяется при плановой замене масла в коробке передач, проводимой 1 раз в 40000 километров. В комплект входят:
• Жидкость для 8-ми и 10-ти ступенчатых АКПП (WSS-M2C949-A) 1L, произв. Ford - 8 канистр
• Фильтр масла для 10-ти ступенчатой АКПП 10R80, произв. Ford - 1 шт
• Прокладка масляного поддона для АКПП 10R80, произв. Ford - 1шт
Комплект применим для 10-ти ступенчатой АКПП 10R80, используемой в следующих моделях Ford:
• Ford Mustang (2015- ) - автомобили, оснащённые 10-ти ступенчатой АКПП 10R80
• Ford F-150 (2015-2020) - автомобили, оснащённые 10-ти ступенчатой АКПП 10R80
• Ford Ranger Mk III (2011 - ) - автомобили, оснащённые 10-ти ступенчатой АКПП 10R80
Количество, требуемое на автомобиль - 1 комплект.
Перед покупкой, обратитесь к нашему специалисту для уточнения типа коробки передач, установленной на Вашем автомобиле.
Необходимо 8 литров масла.
