Marshall представил новые детали для сельхозтехники
Бренд Marshall объявил о расширении фирменного ассортимента расходных частей для сельскохозяйственной техники.
Как сообщает пресс-служба бренда, в продажу поступили 40 новых товарных позиций для техники Lemken, Amazone и Horsch.
Фирменную линейку пополнили детали для обработки почвы: ножи полевой доски, стрельчатые лапы, отвалы предплужника, лемехи, долота и диски бороны.
Запчасти изготовлены из легированной борсодержащей стали марки 30MnCrB5. Некоторые детали имеют наплавку из карбида вольфрама (если предусмотрено заводом-изготовителем) и подходят для любых типов почв, в том числе для плотного грунта.
Материал устойчив к износу: детали, по заявлению производителя, позволяют вспахать 6000 Га земли без потери в качестве.
Предел прочности при растяжении на уровне 1300-1400 МПа обеспечивает повышенную стойкость изделий к образованию трещин и сколов.
