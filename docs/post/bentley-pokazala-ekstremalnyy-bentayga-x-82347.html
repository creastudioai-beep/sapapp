Bentley показала экстремальный Bentayga X

Bentley показала экстремальный Bentayga X
Шоу-кар создан для оценки интереса к возможной серийной версии на фоне падения продаж бренда.
Bentayga X построен на базе версии Speed с V8 мощностью 650 л.с. Дорожный просвет увеличен на 55 мм (310 мм), глубина брода — до 550 мм, колея расширена на 120 мм.
Прототип получил внедорожные шины, расширители арок, экспедиционный багажник с LED-люстрой и доработанную пневмоподвеску.
