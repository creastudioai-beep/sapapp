Посмотрите на первый тюнинг нового Lamborghini Countach

Судя по информации из открытых источников, Lamborghini передала клиентам уже 78 новых Countach LPI 800-4. Из них девять получили прописку в Японии, в том числе это уникальный экземпляр, окрашенный в эффектный колер Blu Hal. Салон «Кунташа» обит красной кожей Rosso Alala, с такой же красной прострочкой и черной декоративной окантовкой. Это первый тюнингованный вариант суперкара Countach LPI 800-4. По крайне мере из тех, которые были показаны публике.
