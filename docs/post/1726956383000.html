Lexus GX 550 получил спецверсию для гольфа
Спецверсия внедорожника создана совместно с компанией Malbon, которая производит экипировку для игры в гольф.
Отличительные особенности Lexus GX 550 Malbon Edition: белые колеса, зеленая окраска кузова с топографической картой полей для гольфа, шкаф со спортивным обмундированием и багажник на крыше.
Рекламный внедорожник выпущен в единственном экземпляре и предназначен для продвижения Lexus на американских турнирах по гольфу.
