Электрический Porsche Cayenne получит беспроводную зарядку
По данным Carscoops, зарядная станция будет устанавливаться на полу, а автомобиль сам опустится на нужную высоту с помощью пневмоподвески для начала зарядки.
Мощность системы составит 11 кВт. Время полной зарядки пока не уточняется.
