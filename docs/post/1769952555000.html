В Японии сделали Nismo-версию для семейного микроа...

В Японии сделали Nismo-версию для семейного микроавтобуса — тюнеры из Dynasty представили бодикит для Nissan Caravan.
Дизайн Alive GT-V вдохновлен Nissan GT-R Nismo, но мощности не прибавилось, потому что улучшения только внешние, а под капотом остался 2,4-литровый мотор на 130 сил.
Комплект улучшений обойдется в ~190 тысяч рублей ($2500).
