Трамп снял ограничения по выбросам для автопрома
Агентство по охране окружающей среды (EPA) упразднила документ Endangerment Finding, принятый при Обаме, который признавал CO2 и другие парниковые газы опасными для климата.
Теперь автопроизводители освобождены от федеральных ограничений на эти выбросы, но нормы по другим вредным веществам остаются.
Отмена включает прекращение налоговых льгот за системы Start-Stop и возврат требований по средней корпоративной топливной экономичности (CAFE) к уровню 2021 года (≈5,8 л/100 км) — без штрафов за несоблюдение CAFE.
Для производителей ДВС это снижает издержки и повышает прибыль, однако климатические активисты ожидают сопротивления на уровне штатов.
