Присядьте: BMW может выпустить свой «ГЕЛИК» — бава...

Присядьте: BMW может выпустить свой «ГЕЛИК» — баварцы уже запатентовали рамную конструкцию.
По слухам, компания работает над большим внедорожником, который станет конкурентом G-Class.
В сети обсуждают проект с индексом G74 — ему приписывают V8 под капотом и полноценную рамную платформу.
Премьера может состояться уже через несколько лет.
