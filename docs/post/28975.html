Каталог Jikiu пополнили семь новых артикулов

Каталог Jikiu пополнили семь новых артикулов

Так, для универсала Nissan Wingroad Y11 стали доступны сайлентблоки задней балки (арт. BH22137).

Владельцам Kia Sorento BL и Sportage бренд предлагает втулки переднего стабилизатора (арт. BL11022), а для родственных Hyundai Accent старше 2011 г. (официально в России не продавался), i20 и Kia Rio для заказа стали доступны наконечники рулевых тяг (левая – арт. ET11046L; правая – ET11046R).

Два артикула пополнили раздел каталога для японских автомобилей. Это проставки пружин для Acura TL (в т. ч. Type S), Honda Accord, Elysion, Inspire и Odyssey (арт. MC28005), а также опоры двигателя для автобуса Nissan Civilian, внедорожников Patrol Y60 и Safari (арт. ME22043).

Для малотоннажника Ford Transit бренд теперь выпускает подвесные подшипники (арт. BC51001).
