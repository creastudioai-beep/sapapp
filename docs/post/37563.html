Немецкая компания Novitec выпустила позолоченный выхлоп для Ferrari

Немецкая компания Novitec выпустила позолоченный выхлоп для Ferrari

Ателье, которое специализируется на доработке в том числе роскошных и ультрасовременных автомобилей, представило программу доработки для двухдверки Ferrari 296 GTB. Автомобиль, ставший первым дорожным спорткаром итальянской марки с двигателем V6 — ранее аналогичный силовой агрегат устанавливался только на формульные болиды, спортпрототипов и машины под маркой Dino — сделали мощнее за счёт минимальных доработок. Традиционно для гибридов тюнеры обратили внимание только на двигатель внутреннего сгорания.
