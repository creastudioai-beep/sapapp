Автодом на колесах для бездорожья за 60 млн. рублей

Автодом на колесах для бездорожья за 60 млн. рублей

EarthRoamer известен тем, что превращает огромные грузовики во внедорожные кемперы. И впервые компания из Колорадо использует шасси Chevy Silverado 6500 HD для своего нового проекта EarthRoamer SX 2023 года.

Массивный дом на колесах оснащен 6,6-литровым турбодизельным двигателем Duramax V8 мощностью 350 л.с. и 700 Нм, работающий с 6-ступенчатую коробку передач. Привод полный

Внутри есть резервуары для пресной воды на 454 литра и необработанной воды на 208 литров. Солнечная батарея мощностью 1600 Вт на крыше питает литий-ионный аккумулятор емкостью 18 000 Втч. В местах, где солнца недостаточно, можно воспользоваться генератором. Два топливных бака вмещают 378 литров дизельного топлива.

Обеденная зона вмещает шесть человек, а двухъярусная кровать четырёх спальная. В ванной комнате есть тропический душ и туалет. Есть шкафы из клена, шкафы из кедра, кожаные сиденья, гранитные столешницы и матрас с эффектом памяти.

Базовая цена EarthRoamer SX составит 995 000 долларов.
