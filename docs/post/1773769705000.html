Новый Volkswagen ID.UNYX 08 — полноразмерный электрический внедорожник с запасом хода 730 км.
Первый полноразмерный электрический внедорожник Volkswagen, разработанный совместно с компанией XPeng в Китае.
📏
Размер
• Длина: 5000 мм
• Ширина: 1954 мм
• Высота: 1672 мм
• Колесная база: 3030 мм
🔋
Аккумулятор и дальность действия
• Литий-железофосфатный аккумулятор CATL
• 82,36 кВт·ч / 95,04 кВт·ч
• Дальность хода: 630 / 700 / 730 км
⚡
Силовой агрегат
• Задний привод: 230 кВт
• Полный привод: 140 кВт + 230 кВт
⚡
Зарядка
• Быстрая зарядка высоким напряжением 800 В
🖥
Технологии интерьера
• Два 14,96-дюймовых дисплея
• 1,74 м² панорамная крыша с интеллектуальным затемнением
• Сиденья из высококачественной кожи наппа
💺
Комфорт
• Электрическое пассажирское сиденье с 14 регулировками
• Подставка для ног + 10-точечный массаж
• Задние откидывающиеся сиденья + подогрев + столики
