Насколько выросли в цене запчасти для Hyundai?

Насколько выросли в цене запчасти для Hyundai?

Эксперты Российского союза автостраховщиков (РСА) зафиксировали 60% подорожание запчастей для Hyundai с марта 2022 года. К примеру, в справочниках цен для ОСАГО дверь багажника Hyundai Santa Fe подорожала почти на 70% — с 49 тысяч до 83,3 тысячи рублей, а впускной коллектор двигателя Solaris с 50,8 тысячи до 83 тысячи рублей. Рост цен связывают с нехваткой запчастей вследствие проблем, с которыми столкнулся импорт после февральских событий.

Наибольший ущерб был нанесен поставкам деталей для корейских, японских и немецких автомобилей. Китайские аналоги пока не могут покрыть весь спрос, создаваемый российскими потребителями. Кроме этого, часть продукции из КНР уступает по качеству запчастям автопроизводителей, которые приостановили работу в России. Однако есть и качественные аналоги, отмечают эксперты, а потому их доля однозначно будет увеличиваться.
