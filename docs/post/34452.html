Ателье Manhart показало комплект тюнинга для купе BMW M4 поколения G82

Ателье Manhart показало комплект тюнинга для купе BMW M4 поколения G82

Взяв за основу версию M4 Competition, которая является флагманом семейства, специалисты построили автомобиль, превосходящий по мощности даже экстремальный BMW M4 CSL — его мощность составляет 635 сил. Теперь у владельцев обычного купе есть весомый козырь перед обладателями лимитированной двухдверки. Главной особенностью тюнингового проекта под названием Manhart MH4 600 стал двигатель.
