Tesla выпустила ПЕРВЫЙ серийный электрокар без рул...

Tesla выпустила ПЕРВЫЙ серийный электрокар без руля и педалей — Cybercab. В машине нет даже зеркал — всё управление берёт на себя система автопилота.
Внутри только экран на панели и минимализм без лишних деталей. Использовать Cybercab планируют в собственном такси-сервисе Tesla.
Ожидаемая цена — около $30 тысяч (примерно ₽2,2 миллиона).
