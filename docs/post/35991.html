Из Mercedes G-Class сделали безумное купе

Из Mercedes G-Class сделали безумное купе

Тюнинг-ателье Mansory переделало Mercedes G-Class в трёхдверное внедорожное купе с необычными открывающимися против хода дверями. Проект получил название Gronos Coupe Evo С.

Салон четырёхместный — передним креслам добавлен механизм складывания, открывающий доступ к двум разделённым центральной консолью сиденьям сзади.

Mansory Gronos Coupe Evo С оснащен 4,0-литровыми 887 сильным двигателем V8 с крутящим моментом 1200 Нм в сочетании с 9-ступенчтаой автоматической коробкой передач.

Всего будет сделано восемь экземпляров трёхдверного внедорожника, а по желанию клиентов автомобиль могут сделать бронированным.
