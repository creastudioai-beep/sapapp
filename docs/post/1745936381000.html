Carville Racing выпустил новую тормозную жидкость
Бренд Carville Racing объявил о расширении фирменного ассортимента тормозных жидкостей.
Для заказа стала доступна полностью синтетическая тормозная жидкость на основе полигликоля с названием DOT 5.1.
Новая тормозная жидкость обладает повышенными антикоррозийными свойствами; DOT 5.1 совместима с эластомерами, сообщает пресс-служба бренда.
Заявленная температура кипения — свыше 260 °С.
Спецификации: FMVSS 116 DOT 5.1, FMVSS 116 DOT 4, FMVSS 116 DOT 3, ISO 4925 Class 5.1, ISO 4925 Class 4, ISO 4925 Class 3, SAE J 1703, SAE J 1704.
Тормозная жидкость DOT 5.1 поставляется в канистрах объёмом 0,5 л. Артикул новинки — L6251426, её розничная стоимость составляет 1230 рублей.
