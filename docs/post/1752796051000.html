Tesla Model 3 получит рекордный запас хода свыше 800 км
Китайский минпром опубликовал документы на новую версию Tesla Model 3+, оснащённую двигателем мощностью 306 л.с. и аккумулятором LG на никель-кобальт-марганцевой основе.
Аналогичная батарея в двухмоторной Model 3 обеспечивает 753 км запаса хода, а версия с одним мотором сможет проехать 800 км по циклу CLTC.
