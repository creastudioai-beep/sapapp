Acura Integra Type S получила заводской тюнинг от Honda Racing Corporation
Автомобиль получил передний бампер с увеличенным сплиттером, задний бампер с диффузором, карбоновый капот, кованые 19-дюймовые диски с шинами Pirelli P Zero Trofeo R. Сзади — углепластиковое антикрыло на алюминиевых стойках.
Мотор у Integra Type S — от Honda Civic Type R. Он выдает 324 л.с. и 420 Нм, работает в паре с шестиступенчатой «механикой».
Чтобы уменьшить массу задние двери заменили на изготовленные из карбона, из него же и сиденья Recaro. Вместо заднего дивана установлена распорка для повышения жёсткости кузова. Подвеска — с койловерами, тормоза — шестипоршневые Brembo.
