RAM готовит дебют первого пикапа с несущим кузовом
Американский RAM запатентовал в США пикап с несущим кузовом — впервые для бренда. В базе данных USPTO появились изображения модели Rampage.
В Stellantis уже рассматривают сборку в Иллинойсе, где ранее выпускался Jeep Cherokee.
Rampage может стать самой доступной моделью RAM.
