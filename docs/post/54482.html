Представили часы, посвященные Honda Type R

Представили часы, посвященные Honda Type R
• Японская часовая компания Casio выпустила новые часы, посвященные "заряженным" автомобилям марки Honda – Edifice Honda Type R Edition.
• Ручные часы пропитаны духом Type R. Casio украсила циферблат культовым логотипом – буква "H" на красном фоне. Дополнительный циферблат имитирует реальный тахометр, на кнопках имеются значки "R", а на петлях для ремешка из алькантары –логотип "Type R".
• Корпус часов изготовлен из смолы, армированной углеродным волокном. Сапфировое стекло устойчиво к царапинам. Edifice Honda Type R Edition обладают водонепроницаемостью до 10 бар (100 метров). Механизму не требуются батарейки, так как в часах используется система солнечной зарядки, обеспечивающая 7 месяцев работы без солнечного света.
• Цена на Casio Edifice Honda Type R Edition в Японии составит около 360 долларов (33,9 тыс. рублей). В продаже будут доступны в следующем месяце.
