Febest расширил фирменный ассортимент газовых упоров
Бренд Febest выпустил новые газовые упоры для капота и крышки багажника.
Шесть новинок уже доступны для заказа, сообщает пресс-служба бренда.
Для багажника Nissan Qashqai J10E выпущен новый упор с артикулом 02117-002.
Для Kia Ceed и Kia Sportage представлены газовые пружины с номерами 22117-001 и 22117-002 соответственно.
Новый упор с артикулом 26117-003 подходит для пятой двери Skoda Yeti. Среди других новинок – упор задней двери с арт. 42117-006 для Lada Niva, упор крышки багажника с арт. 42117-004 для Lada Kalina.
Гарантия на все газовые упоры Febest составляет 2 года без ограничения пробега.
