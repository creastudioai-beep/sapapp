LUZAR представил новые турбокомпрессоры для грузовиков
Бренд LUZAR объявил о расширении фирменного ассортимента турбокомпрессоров для грузовых автомобилей Volvo и DAF.
Две новинки уже доступы для заказа, сообщает пресс-служба бренда.
Турбокомпрессор с артикулом LAT 1010 выпущен для грузовиков Volvo FH и FM (05-) с дв. D13. Тип картриджа – HE500FG.
Деталь с номером LAT 2804 подходит для автомобилей DAF XF 95 (02-) и CF 85 (01-) с дв. XE315C, XE280C и XE355C. Тип картриджа – GT4294.
Ранее бренд выпустил новые радиаторы охлаждения и маслоохладители для грузовиков и автобусов МАЗ.
Радиатор охлаждения LRc 1204 адресован владельцам МАЗ 5340, 6501 и 6512 с дв. Weichai WP7.
Маслоохладитель LOc 1201 представлен для МАЗ 103, 206 и 226 с дв. ММЗ/Mercedes/Deutz/Weichai, а модель LOc 1207 – для МАЗ 103, 203 и 226 с дв. OМ906LA/BF6M1013.
