Kia начала производство Seltos второго поколения
Дебютным рынком для новой Kia Seltos станет Индия: там уже запущено производство кроссовера.
Новый Seltos получил более брутальный дизайн с крупной решёткой радиатора, габариты унифицированы для всех рынков: длина — 4430 мм, колёсная база — 2690 мм.
В салоне — новая архитектура передней панели, современное цифровое табло и расширенный набор ассистентов.
Линейка двигателей включает бензиновые моторы и дизель, все версии для индийского рынка будут переднеприводными. Продажи стартуют весной 2026 года.
