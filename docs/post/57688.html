Уникальный Koenigsegg Jesko с вставками из чистого золота

Уникальный Koenigsegg Jesko с вставками из чистого золота
Один из владельцев редкого Koenigsegg Jesko решил дополнить кузов гиперкара вставками из чистого золота, его озданием занималось ателье Gripen из Швеции.
Кузов автомобиля выполнен из углепластика, который покрыт прозрачным лаком с мелкими золотыми частицами.
Из чистого 24-каратного золота выполнены накладки на антикрыло, насадка выхлопа, декоративные линии на крыше и основания боковых зеркал.
Золотая спецверсия получила название Jesko Odin. Имя заказчика и стоимость гиперкара в компании не раскрывают.
