Коротко о ценах на авто в Китае: Chery выкатила электрокар дешевле ₽700 тысяч — по цене подержанной Гранты.

Внутри всё неожиданно богато — экран на 15,6 дюйма, CarPlay, подогрев и вентиляция сидений. Есть автопарковка и ассистенты для трассы.
