Porsche разработала современную мультимедийку для старых автомобилей

Porsche разработала современную мультимедийку для старых автомобилей

Компания Porsche выпустила мультимедийную систему под названием PCCM Plus, которую можно установить в старые автомобили. Информационно-развлекательный комплекс с поддержкой Android Auto и Apple CarPlay подойдет для спорткара 911 шестого поколения, который выпускали с 2005 по 2008 год, Cayenne первого поколения образцов 2003-2008 годов, а также для Boxster и Cayman, которые сходили с конвейеров с 2005 по 2008 год. Причем новая мультимедийка способна заменить штатные системы без каких-либо модификаций салона или органов управления.
