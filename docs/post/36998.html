Компания Wiesmann представила варианты доработок электрокара Project Thunderball

Компания Wiesmann представила варианты доработок электрокара Project Thunderball

Весной 2022 года Wiesmann представила спорткар, оснащённый парой установленных на задней оси электромоторов суммарной мощностью 680 лошадиных сил. Родстер под названием Project Thunderball, построенный на алюминиевом пространственном каркасе, получил внешние панели из стали, алюминия и углепластика в ретро-стиле. И пока существует одна версия модели, у покупателей есть масса возможностей по персонализации своего электроспорткара. Компания представила три дизайн-проекта, которые могут стать источником вдохновления для клиентов бренда.
