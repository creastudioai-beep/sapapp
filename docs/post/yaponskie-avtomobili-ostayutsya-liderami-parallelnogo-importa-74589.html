Японские автомобили остаются лидерами параллельного импорта

Японские автомобили остаются лидерами параллельного импорта
В июле в Россию было импортировано 28,3 тыс. новых легковушек, из которых 48% — по схемам параллельного импорта. Лидерами среди «альтернативных» поставщиков остаются Китай (47,9%), Киргизия и Беларусь.
Топ-5 моделей по параллельному импорту в июле:
1. Geely Monjaro — 903 шт.
2. Toyota Highlander — 764 шт.
3. Toyota RAV4 — 746 шт.
4. Mazda CX-5 — 688 шт.
5. Toyota Camry — 549 шт.
