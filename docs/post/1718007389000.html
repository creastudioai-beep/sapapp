Пикап Tesla Cybertruck получил версию для полиции
Полицейская модификация разработана тюнинг-ателье Unplugged Performance, которое планирует продавать такие электропикапы охранным агентствам и полицейским департаментам США.
Версия для патрулирования отличается спецсигналами, сиреной, системой громкой связи, панелью управления с дополнительным оборудованием, коваными дисками на 18 дюймов с вседорожной резиной.
