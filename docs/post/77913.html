Dodge вернул легендарный V8 Hemi — только для трека

Dodge вернул легендарный V8 Hemi — только для трека
Dodge представил гоночный Charger Hustle Stuff Drag Pak — лимитированную серию из 50 экземпляров, созданную исключительно для дрэг-рейсинга.
Модель оснащена 5,8-литровым двигателем V8 Hemi третьего поколения с новым алюминиевым блоком, коваными деталями и нагнетателем Whipple.
В пару мотору установлена автоматическая трансмиссия Coan Racing XLT. Автомобиль сертифицирован для заездов NHRA Factory Stock Showdown.
Кузов выполнен из карбона, благодаря чему масса уменьшилась на 45 кг. Установлены новые амортизаторы, стабилизаторы, спортивные сиденья с ремнями и парашют для торможения.
Стоимость гоночного Dodge Charger в США составляет $234 995 (около 19 млн ₽).
