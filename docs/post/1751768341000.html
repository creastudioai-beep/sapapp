Lecar представил новые запчасти для иномарок
В июле Lecar выпустил в общей сложности 107 запчастей для легковых автомобилей зарубежных марок.
В товарной группе «Ступицы» появились 26 новых позиций: 20 ступиц в сборе с подшипниками и шесть подшипников ступиц.
Новинки предназначены для китайских машин, сообщает пресс-служба бренда.
Также в каталоге доступны 16 новых радиаторов, в том числе шесть – для автомобилей из КНР. В товарной группе «Газовые упоры» ассортимент увеличился на 65 позиций.
В настоящее время в ассортименте Lecar представлены 188 деталей для «китайцев». Запчасти бренда подходят для машин марок Chery, Geely, Haval, Changan, Omoda, Exeed, Kaiyi и JAC.
