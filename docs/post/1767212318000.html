Последний экземпляр Karma Revero собран, освобожда...

Последний экземпляр Karma Revero собран, освобождая место для новой модели.
Один из самых необычных автомобилей, появившихся в автомобильном мире в этом столетии, только что был снят с производства. Karma Revero, ранее известный как Fisker Karma, впервые появился в 2011 году. После одной из самых непростых историй, которые только можно себе представить, его нынешние владельцы прекратили его выпуск.
