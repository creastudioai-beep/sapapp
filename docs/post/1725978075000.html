Запчасти для Kia и Hyundai включили в список для параллельного импорта
Минпромторг добавил в список разрешенных товаров для параллельного импорта автозапчасти и компоненты брендов Kia и Hyundai.
Как пояснила пресс-служба министерства, добавление в перечень запчастей корейских брендов обусловлено значительным количеством автомобилей этих марок, используемых российскими гражданами. Оно должно обеспечить внутренний рынок необходимыми запчастями.
