Основные изменения коснулись передней части: уменьшилась радиаторная решетка, изменился дизайн вертикальных элементов, увеличился нижний воздухозаборник.
Главная техническая обновка — появление нового 1.5-литрового турбомотора мощностью 181 л.с. Коробка передач — 7-ступенчатый робот с двумя сцеплениями, привод передний.

Skoda превратила новый Superb в пикап для велогонок
Студенты технологической академии Skoda представили концепт-пикап L&K 130 на базе универсала Superb.
Машина создана для поддержки велогонок, что символично — основатели бренда начинали с производства велосипедов 130 лет назад. Название прототипа подчёркивает эту историю. Skoda также является официальным партнёром Tour de France.
В пикапе багажник переделали в выдвижную грузовую платформу, а правая задняя дверь стала сдвижной для удобства объезда велосипедистов.
В салоне добавили дисплей с телеметрией, новые места для хранения снаряжения и холодильник для воды.
